# CET-6 训练营 🎓

> 基于 React + Vite 构建，AI 驱动，纯前端部署。

面向六级备考的一站式英语训练平台，整合了逐词翻译、语法分析、写作辅助、艾宾浩斯单词复习等核心模块，打开网页即可使用，无需安装任何本地环境。

---

## 功能介绍

### 一、逐词翻译训练

核心思路是：中文和英文的句子结构其实没那么不同，很多时候只要把中文逐词对应翻译过去，句子就已经很地道了。

**怎么用：**

1. 在输入框里写下你想用英文表达的中文句子，比如「政府应该优先改善医疗保障体系」
2. 点击分析，AI 会把这句话拆开，逐词告诉你每个中文词对应的英文是什么、为什么这样对应
3. 对照结果理解句子的构成逻辑，然后自己试着重新翻译一遍
4. 多练几句之后，写作时就会越来越自然，不再绕弯子或堆砌复杂结构

> 建议用自己写作文时真实想表达的句子来练，效果最好。

---

### 二、语法高亮器

输入任意一句英文，AI 会用不同颜色把句子里的语法成分标注出来——主语、谓语、宾语、状语、定语从句等一目了然。

**怎么用：**

1. 把想分析的英文句子粘贴到输入框，点击高亮
2. 对照颜色标注，逐一确认每个部分是什么成分、承担什么语法角色
3. 如果有看不懂的地方，可以点击对应的标注块查看解释

**几个典型用法：**
- 阅读遇到长难句，搞不清楚主干在哪里 → 粘进来直接看
- 自己写完一句话，想检查结构有没有问题 → 粘进来验证
- 把「100句日推」当天的句子粘进来，顺手把语法结构也吃透

---

### 三、写作训练

分两个部分：

**句型模板 + AI 辅助写作**

1. 选择题型（立论驳论 / 问题解决等）
2. 在标题栏输入作文题目，比如「AI 的发展对教育有积极影响，你怎么看？」
3. AI 会按照段落结构给出框架建议——引言怎么写、立论段用什么句式、驳论怎么展开、结尾如何收束
4. 参考建议内容自己填写，不满意可以点击重新生成单个段落

> 建议把 AI 给的内容当参考而不是直接抄，结合自己的思路改写效果更好。

**句式转换训练**

1. 输入一组原始句子
2. AI 会提示如何通过主位推进的方式重新排列，让段落读起来更流畅、更有逻辑感
3. 对照原句和改写后的版本，理解「为什么这样更好」

---

### 四、100句日推 & 单词速查

**100句日推**

基于俞敏洪《100个句子记完5500个六级单词》，用艾宾浩斯 SM-2 算法自动安排复习计划。

1. 每天打开页面，系统会告诉你今天该复习哪几句、该学哪句新的
2. 读完句子后，根据自己的记忆情况点击「记得」「模糊」「忘了」
3. 系统会根据你的反馈自动调整下次复习时间，记得越牢间隔越长，忘了就会加频
4. 坚持每天打卡，不用死记硬背，复习计划会自动帮你安排好

> 学习进度保存在浏览器本地，尽量固定在同一台设备上使用，避免进度丢失。

**单词速查**

在同一个页面的搜索框里输入任意单词，可以查询它是否属于六级词汇，以及查看详细释义。阅读时遇到不认识的词，随手查一下就行。

---

### 五、轩轩小课堂

配套教学文章合集，以博客帖子的形式呈现，目前收录了逐词翻译法、语法嵌套结构、句式重构等专题。

**怎么用：**

1. 进入页面后，选择感兴趣的文章标题点进去阅读
2. 文章里的方法和例句可以结合其他模块来练——看完逐词翻译法的讲解，直接切到「逐词翻译训练」模块动手试
3. 读完之后可以在文章下方留言，记录自己的理解或者问题

## 技术栈

- **框架**：React 18 + Vite
- **样式**：Tailwind CSS
- **路由**：React Router v6
- **状态管理**：Zustand + localStorage 持久化
- **AI**：DeepSeek API（主）/ OpenAI API（备用）
- **Markdown**：react-markdown + remark-gfm
- **部署**：Netlify（静态托管）

## 项目结构

```
src/
├── components/         # 公共组件（Layout 等）
├── data/
│   ├── episodes/       # 轩轩小课堂 Markdown 文章
│   └── cet6Sentences.json   # 100句结构化数据
├── modules/
│   ├── grammar-highlighter/ # 语法高亮器模块
│   └── sentence-transform/  # 句式转换模块
├── pages/              # 页面组件
├── store/              # Zustand 状态
└── utils/              # 工具函数（AI客户端、艾宾浩斯算法等）
scripts/
└── parseSentences.cjs  # PDF 文本 → JSON 解析脚本
```

## 数据说明

本项目是纯前端应用，所有数据（复习进度、写作笔记、单词收藏）都存在浏览器本地，不会上传到任何服务器。清除浏览器缓存会导致数据重置，请注意。
