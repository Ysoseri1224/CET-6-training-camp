# disruption

### rupt

rupt这个词根来自拉丁语rumpere，意思是打破、撕裂。

身边到处都是这个词根：

erupt（爆发）= e（向外）+ rupt，火山把内部的东西向外撕裂出来

corrupt（腐败）= cor（完全）+ rupt，整个内部都被破坏掉了

bankrupt（破产）= bank + rupt，银行账户被"撕票"了，也就是资金链断裂破产没治了

### dis

dis这个前缀你最熟悉的肯定是否定，比如disagree、dishonest。

但它还有"分散、分开"这层意思，也很常见，比如distribute分发。disruption里的dis取的就是"分开"，加上rupt，合起来就是把原本连续的东西强行扯开，所以disruption表示中断、扰乱、颠覆。

有个词叫disruptive technology颠覆性技术，ive就是形容词后缀，disruptive就是颠覆性的，破坏性的

# incurred

incur这个词来自拉丁语，in是进入，cur来自currere，意思是跑、流动。

字面意思就是"跑进去了"——你主动跑进了某个状况里，然后那个状况缠上了你。所以incur的核心含义是"招致、使遭受"，强调是自己的行为导致了某个后果。

incurred是它的过去式/过去分词，常见搭配：

- incur costs / expenses（产生费用）
- incur losses（蒙受损失）
- incur a debt（欠下债务）
- incur someone's anger（惹怒某人）

incur这个词记住一个关键点就够了：这个词天然带着一种"自找的"的语气，所以在描述某人因为自己的决定而承担了某种后果时，用incur会非常准确。你不会说incur happiness，但你可以说incur risks，因为风险是你主动承担的。

incurred是它的过去式，在合同和法律文书里很常见，意思就是"已经产生的/已经招致的"，比如costs incurred（已发生的费用）。

# consensus

### con

con这个前缀来自拉丁语，意思是"共同、一起"，这里再举几个例子：

connect（连接）= con + nect（绑），把大家绑在一起

confirm（确认）= con + firm（坚定），大家一起坚定地认可

conflict（冲突）= con + flict（打击），双方撞在一起互相打击

### sens

sens这个词根能联想到sense就很容易记住了，us也是一种名词后缀。见到us结尾可以先往名词上猜：syllabus、campus、virus，这个us确实很多是从拉丁语直接搬过来的，所以长得比较古典。

consensus = con（共同）+ sens（感受）+ us（名词后缀）

字面意思就是"大家共同感受到的东西"，也就是共识、一致意见。

consensus不等于全票通过，它强调的是大多数人通过沟通和协商达到的共同认可。比如开会讨论到最后"我们达成共识了"，就是We reached a consensus

# competent

### com

com也是"共同、一起"的意思，和con是同一来源，在不同词里拼法略有不同。

### pet

pet这个词根来自拉丁语petere，意思是追求、寻求、冲向。

compete（竞争）= com + pete，大家一起追同一个目标，所以产生竞争

appetite（食欲）= ap + petite，趋向食物的欲望

competent = com + pet（追求）+ ent（形容词后缀）

字面意思是"能够一起去追求、争夺的"，引申为有能力的、胜任的。更直接的逻辑是：能参与竞争，本身就意味着你够格、有实力，所以competent的意思其实是从"有竞争资格"衍生而来的。

用法上来说，competent更偏向"达到了某个标准、具备了完成某件事的基本资格"，带有一种正式认可的意味，比如教师资格证。拿到教资说明你的专业水平是合格的、过关的。competence是它的名词形式，常见搭配有core competence（核心竞争力）。

competent和able、capable这几个词意思接近，但有细微差别，able带点主观能动性，capable更像是一种内化的有能力的感觉。

# resort to

resort这个词的本意是"求助于、诉诸"，to是固定搭配的介词，resort to表示诉诸某种手段，通常是在其他方法都不管用之后的最后选择，带有一定的无奈或迫不得已的语气。

resort to + 名词或动名词（doing）

比如你和室友好好说了很多次，对方就是不配合，最后你不得不去找辅导员，那就是你resort to seeking help from the counselor。

这个词在写作里很好用，当你要描述某人在走投无路的情况下采取了某种行动，resort to就比用would比use要自然得多，而且语气上暗示了"这不是第一选择"，信息量更丰富。

另外resort作名词还有另一个常见意思：度假胜地，比如a beach resort（海滩度假村）。两个意思看起来毫不相关，但其实都有"去那里寻求某种东西"的感觉，一个是寻求解决方案，一个是寻求放松。
