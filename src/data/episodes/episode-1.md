# indicate

### dic

首先是这个dic

dic或者dict这个词根来自于拉丁文dicere，是口述、命令、讲述的意思。因为在古罗马社会主要是通过口述来表态，宣布法律和判决
所以举个例子，predict就是提前说，也就是预测。

indicate，dic

dictionary，dict

ion名词后缀，dic是说，diction，说的词语，说的话，ary→……的集合，把说的内容集合起来就是一本字典

### in

关于in这个词缀，常见的其实不止向内一个意思，还有指向和否定的意思

也就是**1.向内  2.否定  3.指向**

向内很好理解，就是in
指向源自于**index**的拉丁语意思**指向/手指**，比如在数据结构中数组的下标就是index这个单词

```
array=[宝宝，我，好喜欢，你]
```

那么`index（0）`就会**指向**"宝宝"这个元素

所以indicate表示**指向；标示**的时候，in就取指向这个词缀的含义
indicate表示**表明；暗示**的时候，in就取向内这个词缀的含义

### contrast

同时我想到了indicate的反义词contraindicate禁忌
这个词就是源自contrast+indicate，**contrast**就是反面的意思

写作中有个很好用的连接词就是in/by contrast，相比之下；反观，一般用于比较

这个in/by contrast是一种客观对比，强调不同和差异，而On the **contrary**/Contrary to……则是一种完全相反，用于反驳

还有一个词是**opposite**，表示对立面，常见的有on the opposite side或者in the opposite way

oppo+site，site地点，oppo是对立的，反面的

衍伸出来的一个名词也是oppo开头，叫**opponent**反对者，当你想表达某个与你观点相反的人群可以说opponents argue that……

opponents可以代表一类人，在写作中可以代表与你观点相反的人群

引入intro→立论opponents idea→驳论（自己的观点）rebuttal→总结conclusion

xxx的问题在当今社会中被广泛讨论。一类人认为A，然而另一类人认为B。而我认为A or B。

（比如我认为B）

opponents（A的支持者）认为……，+例证或论证

however，moving on to other side（一些转折词），自己的观点B，+与A提出的观点相对应的反驳。

conclusion段，虽然B怎么这么好，但是A更好。（所以，应该怎么这么做）

# recreation

### re

re这个前缀来自拉丁语，最核心的意思是**再一次、重新**，也有**返回**的含义

这个前缀几乎随处可见：

return返回，review重新看→复习/回顾，rebuild = 重新建造→重建

### create

create来自拉丁语*creare*，意思是**生长、产生**，跟中文"创造"几乎是天然对应的

古罗马人用这个词来描述神灵的造物行为——上帝*created*天地，所以create自带一种"从无到有"的感觉

### recreation = re + create的名词

把re + create这两个合在一起，字面意思就是是**重新创造**

那"娱乐/消遣"这个意思是怎么来的呢？中世纪的人们认为，人在工作之后精力耗尽，需要通过玩耍和休息来**重新恢复活力**——也就是把自己"重新创造"一遍，让身心焕然一新

所以recreation最初指的是**恢复精力的活动**，慢慢演变成现在"娱乐、消遣"的意思

# texture

### text

text这个词根也来自拉丁语，意思是**编织、交织**

这里同样是来自古罗马的历史细节。古罗马人在制作文件时，会把字母一行一行地"编织"进羊皮纸或蜡板——就像编布一样，行与行之间紧密交织。所以"文字/文本"这件事，在他们看来本质上就是一种**编织**

这也是为什么：

**text**（文本）= 被编织在一起的文字

**textile**（纺织品）= 被编织出来的布料

**context**（上下文）= con（共同）+ text = 共同编织在一起的内容，也就是语境

### texture = text + ure

**-ure** 是一个名词后缀，表示**状态或结果**，比如nature（自然状态）、structure（建造的状态，也就是结构）。

所以texture字面意思就是编织出来的状态，引申为我们今天说的质地、纹理、手感。

你摸一块布料，感受到它的粗糙或细腻，那就是它的texture，因为布料本来就是编织出来的。

这个逻辑后来延伸到了各种领域：皮肤的texture（肤质），食物的texture（口感），音乐的texture（音质），甚至3D游戏里的texture（贴图/纹理）
